
# Lotto Number Generator

## Overview

A simple web application to generate 6 random numbers from 1 to 45.

## Features

-   Generate 6 unique random numbers.
-   Display the generated numbers.

## Current Plan

-   **index.html**
    -   Add a title for the random number generator.
    -   Add a button to generate numbers.
    -   Add a container to display the numbers.
-   **main.js**
    -   Create a function to generate 6 unique random numbers from 1 to 45.
    -   Display the generated numbers in the UI.
-   **style.css**
    -   Add styles for the title, button, and number container.
